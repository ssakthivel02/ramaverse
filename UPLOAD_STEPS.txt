RamaVerse GitHub Pages Upload Steps

1. Open your GitHub repository: ssakthivel02/ramaverse
2. Upload ALL files and folders from this ZIP contents into the repository root.
3. Do not upload the parent folder itself. index.html must be in the root.
4. GitHub > Settings > Pages > Deploy from branch > main > /(root).
5. Test these URLs:
